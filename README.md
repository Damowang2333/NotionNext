# NotionNext

一个使用 NextJS + Notion API 实现的，部署在 Vercel 上的静态博客系统。为Notion和所有创作者设计。

## 我要如何开始？

只需几分钟即可搭建您的个人站点，欢迎移步[tangly 的博客](https://tangly1024.com/article/notion-next) 查看教程
